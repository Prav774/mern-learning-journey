# Development Environment Setup

## Introduction

Before starting MERN development, it is important to prepare a proper development environment.

This document lists the tools and software required throughout the learning journey.

---

## Operating System

Supported Operating Systems:

* Windows
* Linux
* macOS

For this journey, Windows is being used.

---

## Visual Studio Code

### Purpose

Primary code editor used for development.

### Features

* Syntax Highlighting
* Extensions
* Debugging
* Git Integration
* Terminal Access

### Recommended Extensions

* ES7 React Snippets
* Prettier
* ESLint
* GitLens
* MongoDB for VS Code

---

## Git

### Purpose

Version Control System.

### Benefits

* Track changes
* Manage versions
* Collaborate with others
* Maintain project history

### Verify Installation

```bash
git --version
```

---

## GitHub

### Purpose

Cloud platform for hosting repositories.

### Uses

* Store projects
* Track progress
* Showcase portfolio
* Collaborate with developers

---

## Node.js

### Purpose

JavaScript Runtime Environment.

### Verify Installation

```bash
node -v
```

### Verify npm

```bash
npm -v
```

---

## MongoDB

### Purpose

Database system used to store application data.

### Components

#### MongoDB Server

Stores data locally.

#### MongoDB Compass

Graphical User Interface for managing databases.

---

## Postman

### Purpose

API Testing Tool.

### Features

* Send Requests
* Test APIs
* Debug Responses
* Manage Collections

---

## Google Chrome

### Purpose

Application testing and debugging.

### Developer Tools

Useful tabs:

* Elements
* Console
* Network
* Application
