## Why This Matters

CRUD operations form the foundation of every backend application.

Aggregation helps generate reports and analytics.

Indexes improve query performance and are heavily used in production systems.

## Learning Outcome

After completing this module, you should be able to:

- Perform CRUD operations in MongoDB
- Write complex queries
- Implement pagination
- Build aggregation pipelines
- Use joins with $lookup
- Create indexes for performance optimization
- Understand index tradeoffs
